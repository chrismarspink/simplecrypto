대칭키 파일 암호화
  • 표쥰 AES(Advanced Encryption Standard) 알고리즘을 사용하여 파일을 암호화하며 128/192/256비트 키 길이를 선택할 수 았습니다.

암호 모드(Encryption Mode)

  • ECB: Electronic Code Block Mode
  • CFB: Cipher FeedBack Mode
  • OFB: Output FeedBack Mode
  • CTR: CounTeR Mode
